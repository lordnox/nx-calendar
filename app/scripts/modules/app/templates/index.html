Open Source Angular Calendar Directive

Currently in the works, a complete rewrite and overhaul of angularjs-calendar. The project is dead since I did not contributed anymore. I plan to breath life into it as soon as I find the time.

The project consists of a few directives and a calendar service. Surely I will need to include a connector to recieve google-calendar eventlists. The current work will be visible as a demo on this page. If you need to play around with the base of this project, you can find it here.

Features

  • Workable without service to view your google calendar. (Requires google account)
  • Simple configurable service for angular.js.
  • Customizable templates
  • Year, Month, Day and Eventlist view
